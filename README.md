# Liberty Heart & Vascular Surgery Hospital
## Deploy to Vercel in 30 seconds

### Option 1: Drag & Drop
1. Go to https://vercel.com/new
2. Drag this entire folder
3. Click Deploy ✓

### Option 2: CLI
```bash
npm i -g vercel && vercel --prod
```

### Option 3: GitHub
Push to GitHub → Connect repo at vercel.com → Auto-deploys on push

### Custom Domain
Vercel Dashboard → Settings → Domains → Add `libertyheartsurgery.com`

### Files
- `index.html` — Complete website (zero dependencies, all inline)
- `vercel.json` — Routing + cache headers
